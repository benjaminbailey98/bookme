# **App Name**: Vibe Request

## Core Features:

- Venue Profile Creation: Allows venues to create and manage profiles with company details, addresses, business hours, and contact information.
- Artist Profile Display: Enables users to browse public artist profiles, view pictures, stage names, bios, and performance videos, and select artists for booking requests.
- Subscription Management: Provides subscription management features for venues and artists to pause, resume, and cancel subscriptions. It offers admin tools to process subscription requests and manage payments.
- Intelligent form completion assistance: Utilizes an LLM tool to automatically fill optional parts of booking forms based on entered data or booking history, easing the customer experience and reducing data entry efforts. Provides context-aware suggestions, using entered information to inform the type and depth of details for each category in the form, offering further clarity and minimizing user friction during booking requests.
- Booking Request Submission: Streamlines the process for customers to submit booking requests with event details, venue information, contact information, entertainment preferences, budget, and artist selection.
- Artist Portal Booking Management: Enables artists to manage bookings by changing statuses (pending, approved, denied), updating communication logs, rating events, and adding unavailable dates/times.
- Admin Analytics Dashboard: Provides an admin dashboard with analytics to manage users, artists, venues, bookings, referrals, and subscriptions. Admin can manually enter data and offer subscription benefits.

## Style Guidelines:

- Primary color: A vibrant purple (#9D4EDD) to evoke creativity and energy in the entertainment industry.
- Background color: Light lavender (#E9D5FF), a desaturated tint of the primary purple to give the app a gentle, creative vibe.
- Accent color: A deep indigo (#1B1464) to provide a contrast that draws the user's eye to points of interest and important information.
- Body and headline font: 'PT Sans' for a modern and approachable design that ensures easy readability and engagement.
- Use clean, modern icons that reflect the diversity of entertainment options.
- Design a clean and intuitive layout that facilitates easy navigation and showcases artist profiles effectively.
- Use subtle animations to enhance user interaction and provide visual feedback.
